# 单词按钮与页面重新设计方案

**日期：** 2026-03-28
**状态：** 已确认，待实现

---

## 背景

单词按钮的核心功能是**遮盖 PDF 底图上的单词文本**，并提供点击发音。前一版误将按钮设计为半透明（`rgba` 填充），导致底图文字透出，与功能需求相反。本次重新设计。

---

## 需求确认

| 功能 | 说明 |
|---|---|
| 遮盖文字 | 按钮必须完全不透明，覆盖底图单词 |
| 显示文字模式 | 按钮上显示白色单词文本 |
| 空白遮盖模式 | 按钮为纯色色块，不显示文字（用于自测） |
| 悬停（空白模式）| 变亮但**不显示文字**，保留测试意义 |
| 播放中 | 醒目的橙色 + 脉冲光晕 |
| 答对 / 答错 | 绿色 / 红色，有阴影反馈 |
| 管理员模式 | 半透明橙色虚线框，方便调整位置 |

---

## 方案 A — 经典纯色色块（已选定）

### 单词按钮状态

| 状态 | 背景 | 边框 | 其他 |
|---|---|---|---|
| 默认 | `#3B82F6` | `#2563EB` | — |
| 悬停 | `#60A5FA` | `#3B82F6` | scale(1.05) + 浅阴影 |
| 播放中 | `#EA580C` | `#C2410C` | 脉冲光晕 600ms |
| 答对 | `#16A34A` | `#15803D` | 绿色阴影 |
| 答错 | `#DC2626` | `#B91C1C` | 红色阴影 |
| 管理员 | `rgba(234,88,12,0.15)` | 橙色虚线 | 半透明调位 |

### 文字样式
- 字体：Work Sans，700，白色
- 空白模式下 `.word-text` opacity: 0，悬停也不显示
- `.show-labels` 模式下 opacity: 1

### 圆角
- `border-radius: 6px`（方正感，遮盖彻底）

---

## 页面整体配色

| 区域 | 颜色 |
|---|---|
| 背景 | `#EFF6FF → #DBEAFE` 浅蓝渐变 |
| Header | 白色卡片，圆角 24px |
| 侧边栏 | 白色，蓝色细边框 |
| PDF 容器 | 白色卡片，圆角 18px，蓝色虚线边框 |
| 主按钮 | `#3B82F6` 蓝色系 |
| CTA（播放全部）| `#EA580C` 橙色 |

### 字体
- 标题：Outfit 700
- 正文/按钮：Work Sans 600

---

## 交互规范

- 状态切换：`150ms ease-out`
- 播放脉冲：`0 → 14px spread`，`600ms infinite`
- 悬停位移：`scale(1.05)`，不用 translateY（避免布局偏移）
- 遵守 `prefers-reduced-motion` 媒体查询
